# Evrenzo ERA VR

Kiosk Oculus VR PWA for the Evrenzo at the ERA congress.

### Getting Started

```shell
git lfs pull
npm install
npm start
```

### Running on Oculus

You need to run the app in HTTPS mode for it to work on Oculus:

```shell
git lfs pull
npm install
npm run start-https
```

### Production Build

```shell
git lfs pull
npm install
npm run build
```
